Simple FB Feeds provides immediate access to your Facebook pages, Facebook posts, Facebook news updates, Facebook videos and Facebook photos.

== Description ==

Some of the great features and benefits that will come your way when you download the Simple FB Feeds to your website include:

(i) Show/Hide Frontend view using setting.

(ii) Use Shortcode

Easy to use Different pages or posts or widget using shortcode "[fbf-profile-feeds-display]".

Use This Code in Page Template <code><?php echo do_shortcode('[fbf-profile-feeds-display]'); ?></code>.

iii) Visitors to your site can easily see what is going on over at Facebook. You can easily gain new Facebook fans in this manner, as well as help spread the word about your company.

iv)	Complete documentation.

v)	Gain more Facebook fans and followers.

The Simple FB Feeds can easily be added to your site with all of the above benefits.



== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/directory

2. Activate the plugin

3. You will see Simple Facebook Feeds settings from right side of your wordpress menu.

Settings Configuration:

Facebook Page URL : Put your Facebook page url here. Example: http://facebook.com/facebook

Alignment: You can position your facebook box either on left or right using this option.

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.

That's it!
